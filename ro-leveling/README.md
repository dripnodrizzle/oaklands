# RO-LEVELING Scripts

Scripts specifically designed for the RO-LEVELING Roblox game.

## Available Scripts

- **ro-leveling-infinite-rolls.lua** - Gives infinite dodge rolls by exploiting AddRolls remote
- **ro-leveling-decompiled-roll.lua** - Decompiled roll mechanic for analysis

## Usage

Load scripts via your Roblox executor (Synapse, KRNL, etc.)

## Notes

RO-LEVELING uses a dodge/roll system with:
- `ReplicatedStorage.Global.RollEvent` - Handles roll execution
- `ReplicatedStorage.Global.AddRolls` - Adds roll charges to player
